# App Store Territory Codes

Complete list of all 175 App Store territories with their codes, currencies, and regions.

## Territory Code Format

- **iTunes API:** Two-letter ISO 3166-1 alpha-2 (e.g., `us`, `gb`, `de`)
- **App Store Connect API:** Three-letter ISO 3166-1 alpha-3 (e.g., `USA`, `GBR`, `DEU`)

## Currency Considerations

### Stable Currencies (Less Frequent Adjustments)
- USD, EUR, GBP, CHF, JPY, AUD, CAD, SGD

### Volatile Currencies (Monitor Closely)
- TRY (Turkish Lira) — High inflation
- ARS (Argentine Peso) — Frequent devaluations
- NGN (Nigerian Naira) — Recent volatility
- EGP (Egyptian Pound) — Devaluation risk
- PKR (Pakistani Rupee) — Economic instability

### Dollarized Economies
These use USD directly:
- Ecuador, Panama, El Salvador, Cambodia, Zimbabwe
